## String Manipulation and Algorithms (Part 1)

### 1. **Core String Concepts (Java)**
    - [ ] **Immutability:** `String` objects are immutable.
    - [ ] **String Pool:** How Java stores and reuses string literals.
    - [ ] **`String` vs. `StringBuilder` vs. `StringBuffer`:**
        - [ ] `String`: Immutable, best for non-changing strings.
        - [ ] `StringBuilder`: Mutable, not thread-safe. Use for building strings in a single thread.
        - [ ] `StringBuffer`: Mutable, thread-safe. Slower than `StringBuilder`.
    - [ ] Common `String` methods: `charAt()`, `length()`, `substring()`, `equals()`, `hashCode()`, `split()`, `toCharArray()`.
